Simallo.com

As the lead developer, I developed Simallo.com from concept to deployment. Read more...

Autism Symptom Tracker

I designed and built a desktop Windows form application to help users track autism-related symptoms, patterns, and daily mood/behavior changes. Read more...

The application included a daily questionnaire for users & caregivers, which is saved to a secure SQLite database enabling encrypted storage and CRUD operations. I built dynamic features with C# to analyze the data over time, creating custom advice and generating charts to visualize symptom severity. Additional features include a reference page with FAQs from peer-reviewed articles and an organized journal entry system. Throughout the project, I focused on accessibility, clean UI/UX, and real-world usability for neurodivergent individuals and caregivers, writing clear, maintainable code for future enhancements.

Express.js Calorie Tracker

I developed a full-stack web application with Express.js and Node.js that allows users to track their daily food intake and total calories consumed. Read more...

The app features a dynamic tallying system that displays the total calories consumed in real-time as users add food items. Users can set a daily calorie goal to visually compare their consumption and track their progress. To ensure data persistence, the application is connected to a MongoDB database, which keeps a historical record of previous days' calorie counts and tracks how consistently the user has adhered to their goals over time.
