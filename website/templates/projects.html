Autism Symptom Tracker

I designed and built a desktop Windows form application to help users track autism-related symptoms, patterns, and daily mood/behavior changes. Read more...

The application included a daily questionnaire for users & caregivers, which is saved to a secure SQLite database enabling encrypted storage and CRUD operations. I built dynamic features with C# to analyze the data over time, creating custom advice and generating charts to visualize symptom severity. Additional features include a reference page with FAQs from peer-reviewed articles and an organized journal entry system.
